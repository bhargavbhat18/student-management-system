# 🎓 Student Management System

A simple and responsive Student Management System built using **Spring Boot**, **React**, and **MySQL**. This project was created to learn full-stack development and understand how frontend and backend applications communicate using REST APIs.

The application allows users to add, view, edit, and delete student records through an easy-to-use interface.

---

## 🚀 Features

* Add new student records
* View all students
* Update student information
* Delete student records
* Responsive and clean user interface
* REST API integration between frontend and backend

## ⚙️ Getting Started

### Clone the repository

```bash
git clone https://github.com/bhargavbhat18/student-management-system.git
```

### Backend Setup

1. Open the **backend** folder in IntelliJ IDEA.
2. Create a MySQL database (for example, `studentdb`).
3. Update the database details in `application.properties`.
4. Run the Spring Boot application.

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

The application will start on your local machine, and the frontend will communicate with the Spring Boot backend through REST APIs.

## 📌 API Operations

* **GET** – Retrieve all students
* **GET** – Retrieve a student by ID
* **POST** – Add a new student
* **PUT** – Update student details
* **DELETE** – Remove a student
